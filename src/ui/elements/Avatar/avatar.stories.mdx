The avatar element is designed to be used both for individuals and groups agent.

## Quick start
To get started with the avatar component, you just need to import it.

```jsx
import Avatar from 'src/ui/Avatar'

<Avatar size="m" variant="rounded" src="https://picsum.photos/id/342/50/50" initials="AA" />
```
